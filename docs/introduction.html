IntelliJ V 0.0.1-alpha Help

Introduction

IntelliJ V is an open source plugin for V compatible with all JetBrains IDEs. You can find the plugin source code in github repository.

This documentation is structured to guide you through the basics of using the plugin to advanced features. Start with Quick Start Guide or FAQ and then move on to the others that interest you.
